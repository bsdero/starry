# My Tools

### 🛠️ System & File Management
*   **`bash`**: Execute shell commands in a secure environment.
*   **`read`**: Read the contents of files or list directory entries.
*   **`write`**: Create new files or overwrite existing ones.
*   **`edit`**: Replace specific text within a file.
*   **`glob`**: Search for files using pattern matching (e.g., `**/*.py`).
*   **`grep`**: Search for specific text or regex patterns within files.

### 🌐 Information & Integration
*   **`webfetch`**: Retrieve content (text or JSON) from a specific URL.
*   **`skill`**: Load specialized workflow instructions for specific tasks.

### 🤖 Task & Agent Coordination
*   **`task`**: Launch autonomous subagents (**Assistant**, **Coder**, or **Researcher**) to handle complex, independent sub-tasks.
*   **`todowrite`**: Maintain and update a persistent todo list to track progress.
*   **`question`**: Formally ask you for clarification or missing information before proceeding.
